## 🚀 Getting Started

### View Locally
```bash
# Just open the file in any browser
open devamjitsinh_portfolio.html

# Or via VS Code Live Server
# Right-click → Open with Live Server
```

### Host on GitHub Pages
```bash
# 1. Create a new GitHub repository
git init
git add devamjitsinh_portfolio.html
git commit -m "🚀 Initial portfolio launch"
git branch -M main
git remote add origin https://github.com/Devam665/<repo-name>.git
git push -u origin main

# 2. Go to repo Settings → Pages
# 3. Source: Deploy from branch → main → / (root)
# 4. Your site will be live at:
#    https://Devam665.github.io/<repo-name>/
```

### Host on Netlify (Drag & Drop)
1. Go to [netlify.com](https://netlify.com) → Log in
2. Drag and drop `devamjitsinh_portfolio.html` onto the deploy area
3. Your site is live in 30 seconds with a free URL
